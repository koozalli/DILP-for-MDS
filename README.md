# DILP-for-MDS
source code of the ADCOM 2025 paper: Distributive Integer Linear Programming for Minimum Dominating Set Problem
---------------------------------------------------------------------------------------------------------------
This is the source code of the paper "Distributive Integer Linear Programming for Minimum Dominating Set Problem", which has been accepted for presentation in the 30th International Conference on Advanced Computing and Communications ([ADCOM 2025](https://accsindia.org/adcom2025/)) at the International Institute of Information Technology Bangalore (IIIT-Bangalore) during 17th-20th December 2025, in association with the Advanced Computing and Communications Society (ACCS).

https://accsindia.org/adcom-2025-schedule/

This paper has been written by Rabea Mazen Saleh.

The authors of the idea are Rabea Mazen Saleh and Dr. Dharmendra Prasad Mahato

The author of the source code is Rabea Mazen Saleh
